Requirements!

Please note that this theme is compatible with phpBB version 3.2.2

Your hosting provider must have at least php version 7 (recommended 7.3).

To save you some time to search for phpBB installer we added here a link to donwload phpBB 3.2.2

Also you will need a FTP client (use your favorite client, in this documentation i will use WinSCP)
