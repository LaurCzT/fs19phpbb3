# Farming Simulator 19 Forum Theme
This is a theme for phpbb3 (version 3.2.2). This theme is clone 1:1 of the official one.
I made this theme for those who want's to create their Farming Community.

To install theme please read the documentation provided here.
You will find a link from official site to download phpbb3 version 3.2.2 in documentation.

If you need more asistance with theme instalation contact me via github!
